## useEffect란?

`useEffect`는 컴포넌트의 **사이드 이펙트**를 처리하는 Hook입니다. API 호출, 구독, DOM 조작 등에 사용합니다.

```tsx
useEffect(() => {
  // 사이드 이펙트 실행
}, [dependencies])
```

## 의존성 배열

의존성 배열에 따라 실행 시점이 달라집니다.

```tsx
// 매 렌더링마다 실행
useEffect(() => { ... })

// 마운트 시 1회만 실행
useEffect(() => { ... }, [])

// count가 변경될 때만 실행
useEffect(() => { ... }, [count])
```

## 클린업 함수

구독 해제, 타이머 정리 등에 사용합니다.

```tsx
useEffect(() => {
  const timer = setInterval(() => {
    console.log('tick')
  }, 1000)

  return () => clearInterval(timer) // 클린업
}, [])
```
