## 첫 번째 포스트

이 블로그는 **Next.js v15**와 **Tailwind CSS v4**로 구축되었습니다.

GitHub Pages에 정적 사이트로 배포되며, MDX를 사용하여 글을 작성합니다.

### 기술 스택

- **Framework:** Next.js v15 (Static Export)
- **Styling:** Tailwind CSS v4
- **Content:** MDX + Velite
- **Deploy:** GitHub Actions → GitHub Pages

### 코드 하이라이팅 예시

```typescript title="hello.ts" {3}
function greet(name: string): string {
  console.log('Hello!')
  return `Hello, ${name}!`
}
```
